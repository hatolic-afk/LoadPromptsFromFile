================================================================================
                    HATOLIC COMFYUI NODES
                    Текстовый файл → CONDITIONING
================================================================================

Просто читает промпты из txt файла и превращает в CONDITIONING для ComfyUI.
Никакого ИИ, никакой магии - просто текст в кондишен.

================================================================================
БЫСТРЫЙ СТАРТ
================================================================================

1. Кидаешь файл prompts.txt в папку ComfyUI/input/
2. Вставляешь нод в воркфлоу
3. Подключаешь CLIP
4. Profit

================================================================================
ЧТО УМЕЕТ
================================================================================

✅ Читает промпты из txt файла
✅ Три режима: sequential, random, by_number
✅ Автосброс счетчика при смене параметров
✅ Поддерживает UTF-8 и Windows-1251
✅ Игнорирует комментарии (#) и пустые строки

================================================================================
ПАРАМЕТРЫ
================================================================================

prompts_file      - твой файл с промптами (из папки input/)
selection_mode    - sequential | random | by_number
prompt_number     - с какого номера стартовать (sequential) или конкретный номер
shuffle_each_time - перемешивать перед выборкой (для random)
seed              - для воспроизводимости случайных выборок

================================================================================
ФОРМАТ ФАЙЛА
================================================================================

Обычный txt файл с промптами:

1. Первый промпт
2. Второй промпт
3. Третий промпт

Можно с комментариями:

# Это комментарий
1. Текст промпта
2. Еще промпт

================================================================================
ПРИМЕР ВОРКФЛОУ
================================================================================

CLIP Loader → TextFile Prompt → KSampler
                    ↓
               Prompt Text (для информации)

================================================================================
РЕЖИМЫ
================================================================================

sequential - берет по порядку: если указал 10, то 10, 11, 12...
random      - случайный промпт
by_number   - всегда конкретный номер

Счетчик сбрасывается при изменении любого параметра.

================================================================================
ИСПРАВЛЕНИЕ КОСЯКОВ
================================================================================

Если счетчик не сбрасывается - измени любой параметр (prompt_number, 
selection_mode, seed) и он сбросится.

Если файл не видно - перезапусти ComfyUI после добавления файла в input/.
